# 🏨 **Endless Hotel** 🏨
### 📒 **스토리**
> **1901년, 이탈리아. 한 인물이 외딴 시골 호텔에서 눈을 뜬다. 곧 호텔 곳곳에서 기묘한 이상 현상이 나타나고,**  
> **그 원인을 해결하기 위해 마지막 로비 층에 도달해야만 한다.**  
> **그러나 유일한 이동 수단인 엘리베이터는 반복되는 오류와 함께 제대로 작동하지 않는다.**


**“당신은 이 미로 같은 호텔을 탈출할 것인가, 아니면 끝없는 루프에 갇힌 희생자가 될 것인가?”**

### 📢 **소개**
-> Endless Hotel은 끝없이 반복되는 호텔에 갇힌 투숙객이 탈출을 하는 게임입니다!

### 🧭 **기획 배경**
기존의 8번 출구류 공포 게임을 기반으로, 나폴리탄 괴담이라는 서사를 결합해 새로운 분위기의 호러 경험을 제공하고자 했다.  
기존 팀들이 시도하지 않던 독창적 소재와 예측 불가한 시스템을 도입함으로써, 장르적 한계를 넘어서는 실험적 게임을 기획하게 되었다.  

### 🗝️ **주요 특징**
_`#심리적공포` `#퍼즐` `#초현실적` `#분위기있는` `#3D`_
1. 리미널 스페이스: 끝없이 반복된 기묘한 공간 구조로 심리적 공포를 극대화
2. 다양한 이상현상: 게임을 진행하며 마주하는 예측불가한 이상현상
3. 규칙 괴담: 나폴리탄 괴담 형식의 게임 진행 방식
4. 몰입도 높은 연출: 역동적인 1인칭 모션과 캐릭터, 3D 입체 사운드
5. 입체감 있는 스토리: 정신병을 앓고 있는 건축가의 이야기
